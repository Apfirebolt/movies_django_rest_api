About Django Movies API

Django Movies API is a sample project demonstrating how to build a RESTful API using Django. This project is designed to be extended with front-end frameworks like React, Angular, or Vue.


This project includes various features such as user authentication, movie listings, and reviews. It serves as a great starting point for developers looking to build their own movie-related applications.
